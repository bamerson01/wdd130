Web Developement

As I mentioned on my home page, my hobby in computer music led me into a new passion, computer programming. How? Well, I like to use a program called Ableton Live to make and record music.

I have been using this program for 20 years. In the early years, it was not as feature robust, but they provided access to a program called Max for Live, a graphical coding environment, sort of like a drag and drop code environment that allowed musicians to build devices to solve issues or create more efficient workflows when making music. I often found myself noodling around and building devices way more than I was making music. So here I am, enrolled at BYUi to pursue a degree in Web Development.

Now I am taking a course on web fundamentals and learning more about HTML and CSS. This website is my final project. I am also taking a course on Linux and learning how to navigate and create scripts in the shell command line.

I have learned a lot so far and am looking forward to investing more of my time on this journey of learning. I will update this site with more info as I progress. I want to demonstrate and showcase my newly acquired skills. Here are just a few topics I know:
